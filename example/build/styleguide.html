#TabContainer

src/components/tab-container/tab-container.vue

The TabContainer will add tabs. Each tab wil load another component.

Example tabs value: [{ name: "PROFILE", content: odEmployeeProfile }, { name: "PROJECTS", content: odEmployeeProjects }] The name is title of the tab and the content is a component which will be shown when the tab is active.

Props

Name Type Default Description
tabs Array Required Array with Javascript Objects.

Import

import TabContainer from "components/tab-container/tab-container.vue";
